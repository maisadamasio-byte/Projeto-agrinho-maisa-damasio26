#Maisa_Damasio_Projeto_Agrinho_2026

O objetivo deste site é conscientizar as pessoas sobre a importância da sustentabilidade, mostrando como pequenas atitudes podem ajudar a preservar o meio ambiente. Além disso, o site apresenta uma forma simples de calcular um índice de sustentabilidade, incentivando o uso responsável dos recursos naturais para construir um mundo melhor
